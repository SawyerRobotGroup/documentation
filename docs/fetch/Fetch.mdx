## Fetch Robot Introduction

We use a Fetch robot from Fetch Robotics who is also known as Qiborg.

This robot had a variety of different features

- Jointed Arm
- Sensors

### Some Basic Information
Set ROS master as Fetch: (need to do this in every terminal that you want ROS to talk to Fetch)
`export ROS_MASTER_URI="http://{hostname or ip}:11311"`
Set your own ROS_IP
`export ROS_IP=YOUR_IP_HERE`

Troubleshoot with 
`roswtf`
