## Overview

MoveIt is a software that is used for calculating and planning complex paths and is how we get Sawyer to grab the blocks that it sees from the vision subsystem.

There are some tutorials on [rethinkrobotics.com](https://rethinkrobotics.com) which are very helpful:

- [MOVEIT TUTORIAL](https://sdk.rethinkrobotics.com/intera/MoveIt_Tutorial)
- [MOTION INTERFACE TUTORIAL](https://sdk.rethinkrobotics.com/intera/Motion_Interface_Tutorial)
