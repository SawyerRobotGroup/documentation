## Overview

There are many tutorials for learning the ins and outs of the Sawyer Robot [here](https://sdk.rethinkrobotics.com/intera/Tutorials) but to get you started here is a list of important ones:

- [Hello Robot](https://sdk.rethinkrobotics.com/intera/Hello_Robot) (Simple overview of many topics and a good place to start)

- [HEAD DISPLAY IMAGE EXAMPLE](https://sdk.rethinkrobotics.com/intera/Head_Display_Image_Example)

- [HEAD MOVEMENT EXAMPLE](https://sdk.rethinkrobotics.com/intera/Head_Movement_Example)

- [GRIPPER EXAMPLE](https://sdk.rethinkrobotics.com/intera/Gripper_Example)

- [JOINT POSITION EXAMPLE](https://sdk.rethinkrobotics.com/intera/Joint_Position_Example)

- [JOINT POSITION WAYPOINT EXAMPLE](https://sdk.rethinkrobotics.com/intera/Joint_Position_Waypoint_Example)

- [INTERACTION CONTROL TUTORIAL](https://sdk.rethinkrobotics.com/intera/Interaction_Control_Tutorial)

Going over these examples should give you a basic idea of how the Sawyer Robot works and will allow you to understand [MoveIt](tiborg/MoveIt.md) much better.
